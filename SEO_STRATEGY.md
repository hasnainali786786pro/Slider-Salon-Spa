## 6. Technical & Performance SEO Checklists

Google’s Page Experience signals penalize slow websites. Ensure the React frontend meets strict benchmarks.

*   [x] **Next-Gen Images:** Fully compress and host all gallery/service imagery as WebP format with strict size dimensions to prevent Layout Shift (CLS).
*   [x] **Lazy Loading Assets:** Apply `loading="lazy"` or Framer Motion layout deferrals on below-the-fold component blocks.
*   [x] **Mobile Responsiveness:** Strict horizontal scroll boundaries, custom touch targets (>44px), and responsive padding classes.
*   [x] **Resource Preconnects:** Preconnect to typography foundations (`fonts.googleapis.com`) for minimized font render blocking.
*   [x] **Semantic Document Structure:** Keep elements framed strictly using `<nav>`, `<header>`, `<main>`, `<section>`, `<footer>` instead of un-nested div structures.
